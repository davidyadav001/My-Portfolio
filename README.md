# Portfolio Website

A modern, responsive portfolio website showcasing professional experience, skills, and projects.

## About

This is a personal portfolio website for **Dharmendra Yadav**, a Full Stack Developer specializing in React.js, React Native, Node.js, and modern web technologies.

## Features

- **Responsive Design** - Works seamlessly on all devices
- **Modern UI/UX** - Clean and professional design
- **Smooth Animations** - Enhanced user experience with WOW.js animations
- **Contact Form** - Integrated contact form for inquiries
- **Dynamic Content** - Experience years calculated automatically

## Technologies Used

- HTML5
- CSS3 (Custom styling with SCSS-like structure)
- JavaScript (jQuery)
- Bootstrap 4
- Font Awesome Icons
- WOW.js (Scroll animations)
- Typed.js (Typing animation)
- Owl Carousel
- Lightbox

## Sections

- **Hero** - Introduction and main heading
- **About** - Professional summary and skills
- **Services** - Services offered
- **Experience** - Work history and internships
- **Contact** - Contact form

## Setup

1. Clone or download this repository
2. Open `index.html` in a web browser
3. For contact form functionality, ensure PHP is configured on your server

## Customization

- **Theme Color**: Edit `#6e504e` in `css/style.css` to change the primary theme color
- **Content**: Update personal information in `index.html`
- **Images**: Replace images in the `img/` folder
- **Experience Years**: Modify the start year in `js/main.js` (line 108)
